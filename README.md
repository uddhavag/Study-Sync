## Installation
1. Clone the repository:
   ```sh
   git clone https://github.com/uddhavag/Study-Sync.git
   cd Study-Sync
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the backend server:
   ```sh
   cd backend
   npm start
   ```
4. Start the frontend server:
    ```sh
   cd frontend
   npm run dev
   ```
